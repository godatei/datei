Welcome to the Datei blog!

Datei is an open source, self-hosted file management platform. The source code
is available on [GitHub](https://github.com/godatei/datei).

## What to expect from the Datei blog

We will use this blog to share release announcements, tutorials, and insights
into how Datei is built.

## How to get started with Datei

Head over to the [documentation](/docs/) to learn more about Datei and how to
run it yourself.
